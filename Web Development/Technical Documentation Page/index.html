Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects; for example:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.

  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server.

What should you know already?

This guide assumes you have the following knowledge:

  • A basic understanding of the Internet and the World Wide Web (WWW).

  • Knowledge of HyperText Markup Language (HTML).

  • Some prior programming experience.

Writing Hello World

To begin, first open the Scratchpad and write your first "Hello World" JavaScript code:

function greet(name) {
alert("Hello " + name);
}
greet("World");

Select the code in Scratchpad and press Ctrl+R to watch it run in your browser!

Declaring Variables

You can declare a variable in three different ways:

  1. With the keyword var. For example, var x = 37. This syntax can be used to declare both local and global variables.

  2. By simply giving a variable a value. For example, x = 37. This always declares a global variable. It generates a strict JavaScript warning. You shouldn't use this variant.

  3. With the keyword let. For example, let j = 22 or t = (10*2). This syntax can be used to declare a block scope local variable.

Data Types

The latest standard defines seven data types:

  • Six data types that are primitive types:

    • Boolean. true and false.
    • null. A special keyword denoting a null value. Because JavaScript is case-sensitive, null is not the same as Null, NULL, or any other variant.
    • undefined. A top-level property whose value is undefined.
    • Number. 42 or 3.14159.
    • String. "Hello!"
    • Symbol (new in 2015). A data type whose instances are unique and immutable.

  • and Object.

Although there are not many types, they still enable you to perform useful functions within your applications. Objects and functions are the other fundamental elements in the language. You can think of objects as named containers for values, and functions as procedures that your application can perform.
